- 테이블 나눠놨는데 합쳐서 봐야 할 때

예시 테이블

**user**

|id|name|
|---|---|
|1|철수|
|2|영희|
|3|민수|

**order**

|id|user_id|item|
|---|---|---|
|1|1|키보드|
|2|1|마우스|
|3|2|모니터|

---

**INNER JOIN** - 양쪽 다 있는 것만
```sql
SELECT user.name, order.item
FROM user
INNER JOIN order ON user.id = order.user_id;
```
결과:

|name|item|
|---|---|
|철수|키보드|
|철수|마우스|
|영희|모니터|
