## Key
- 데이터를 구분하고 테이블끼리 연결 하기 위한 기준
---

### PK (Primary Key) - 기본키
- 각 행을 유일하게 식별
- 중복 X, NULL X
- 테이블당 하나만 가능
```sql
CREATE TABLE user (
	 id INT PRIMARY KEY,
	 name VARCHAR(50) 
 );
```

### FK(Foreign Key) - 외래키
- 다른 테이블의 PK를 참조
- 테이블 간 관계 연결용
```sql
CREATE TABLE order ( 
	id INT PRIMARY KEY, 
	user_id INT, 
	FOREIGN KEY (user_id) REFERENCES user(id) 
);
```

### UK (Unique Key) - 유니크 키 
- 중복X ,NULL은 허용(DB마다 다름)
- PK 아닌데 유일해야하는값
```sql
email VARCHAR(100) UNIQUE
```

**정리**

| 키   | 중복  | NULL | 용도     |
| --- | --- | ---- | ------ |
| PK  | ❌   | ❌    | 고유키,   |
| FK  | ⭕   | ⭕    | 테이블 연결 |
| UK  | ❌   | ⭕    | 유일성 보장 |
