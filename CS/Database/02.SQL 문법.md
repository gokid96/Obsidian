### CRUD  - 기본 4가지 연산

### SELECT (조회)
```sql
-- 전체 조회
SELECT * FROM user;
-- 특정 컬럼
SELECT name,email FROM user;
-- 조건 조회
SELECT * FROM user WHERE id = 1;
```
---
### INSERT (삽입)
```sql
INSERT INTO user (name, email)
VALUES ('철수', 'a@a.com');
```
---
### UPDATE (수정) 
```sql
UPDATE user
SET name = '영희'
WHERE id = 1;
--WHERE 없으면 테이블 전체 수정
```
---
### DELETE (삭제)  WHERE 없으면 전체 삭제
```sql
DELETE FROM user WHERE id = 1;
--WHERE 없으면 테이블 전체 삭제
```
