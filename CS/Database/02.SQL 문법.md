### CRUD  - 기본 4가지 연산

### SELECT (조회)
```sql
-- 전체 조회
SELECT * FROM user;
-- 특정 컬럼
SELECT name,email FROM user;
-- 조건 조회
SELECT * FROM user WHERE id = 1;
```
---
### INSERT (삽입)
```sql
INSERT INTO user (name, email)
VALUES ('철수', 'a@a.com');
```
---
### UPDATE (수정) 
```sql
UPDATE user
SET name = '영희'
WHERE id = 1;
--WHERE 없으면 테이블 전체 수정
```
---
### DELETE (삭제)  WHERE 없으면 전체 삭제
```sql
DELETE FROM user WHERE id = 1;
--WHERE 없으면 테이블 전체 삭제
```
---
### 정렬 & 필터링
```sql
-- 정렬
SELECT * FROM user ORDER BY name ASC;   -- 오름차순
SELECT * FROM user ORDER BY name DESC;  -- 내림차순
-- 개수 제한
SELECT * FROM user LIMIT 10;
-- 조건 여러개
SELECT * FROM user WHERE age > 20 AND name = '철수';
```
---
### GROUP BY & 집계함수
```sql
--부서별 인원수
SELECT dept, COUNT (*)
FROM employee,
GROUP BY dept;
--집계함수: COUNT, SUM, AVG, MAX, MIN
```
---

### HAVING
- GROUP BY 결과에 조건 걸때 (WHERE은 그룹전에 필터링)
```sql
SELECT dept, COUNT(*)
FROM employee
GROUP BY dept
HAVING COUNT(*) >= 5; -- 5명 이상인 부서만
```
