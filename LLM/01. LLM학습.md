LLM 학습의 큰 그림
```
┌──────────────────────────────────────┐
│           딥러닝 (Transformer)        │
│                                      │
│   1. Pre-training (사전학습)          │
│          ↓                           │
│   2. Fine-tuning (파인튜닝)           │
│       ├── SFT                        │
│       ├── RLHF                       │
│       └── DPO 등                     │
│                                      │
└──────────────────────────────────────┘
```

| 구분               | 설명                                       |
| ---------------- | ---------------------------------------- |
| **딥러닝**          | LLM 전체를 관통하는 기반 기술 (특히 Transformer 아키텍처) |
| **Pre-training** | 대규모 데이터로 언어 자체를 학습                       |
| **Fine-tuning**  | 목적에 맞게 조정 (SFT, RLHF, DPO 등이 여기 포함)      |
