## 정의
"단계별로 생각해봐" 프롬프트로 모델의 추론 과정을 유도하는 기법

## 효과
- 수학, 논리, 상식 추론 문제에서 정확도 대폭 향상
- GPT-4 기준 일부 벤치마크에서 10~20% 향상

## 종류

### Few-shot CoT
추론 과정 예시를 함께 제공
```
Q: 가게에 사과 23개가 있었습니다. 5개를 팔고 8개를 더 들여왔습니다. 
   지금 사과는 몇 개입니까?

A: 차근차근 생각해봅시다.
   1. 처음에 23개가 있었습니다.
   2. 5개를 팔았으니 23 - 5 = 18개
   3. 8개를 들여왔으니 18 + 8 = 26개
   따라서 답은 26개입니다.

Q: (새 문제)
```

### Zero-shot CoT
"Let's think step by step" 한 줄만 추가
```
Q: (문제)
A: Let's think step by step.
```

이것만으로도 효과 있음 (2022년 논문)

## 코드 예시
```python
# Zero-shot CoT
prompt = f"""
{question}

Let's think step by step.
"""

# 또는 한국어로
prompt = f"""
{question}

단계별로 생각해봅시다.
"""
```

## 언제 쓰면 좋나
- 수학 문제
- 논리 퍼즐
- 다단계 추론
- 복잡한 분석

## 언제 안 써도 되나
- 단순 사실 질문 ("서울의 인구는?")
- 창작 작업
- 분류 작업 (간단한 경우)

## 주의점
- 토큰 사용량 증가 (추론 과정도 출력하니까)
- 간단한 문제에는 오히려 비효율
