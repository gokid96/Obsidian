## 정의
파인튜닝 없이 프롬프트에 예시를 넣어서 모델이 패턴을 파악하게 하는 방식

## 핵심
- 모델 가중치는 변하지 않음
- 프롬프트만으로 "학습"하는 것처럼 동작

## 종류

### Zero-shot
예시 없이 바로 질문
```
Q: 이 문장의 감정은? "오늘 기분 최고야"
A: 긍정
```

### One-shot
예시 1개 제공
```
예시) "짜증나" → 부정

Q: "오늘 기분 최고야" → ?
A: 긍정
```

### Few-shot
예시 여러 개 제공
```
예시1) "짜증나" → 부정
예시2) "좋아요" → 긍정
예시3) "그냥 그래" → 중립

Q: "오늘 기분 최고야" → ?
A: 긍정
```

## 장점
- 파인튜닝 없이 빠르게 적용
- 예시만 바꾸면 다른 작업도 가능

## 한계
- 컨텍스트 길이 제한
- 복잡한 작업은 예시만으로 부족
