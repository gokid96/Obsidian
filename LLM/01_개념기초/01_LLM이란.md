## 정의
- Large Language Model, 대규모 언어 모델
- 수십억~수천억 개 파라미터로 텍스트의 다음 단어를 예측하도록 학습
- 핵심 원리: "주어진 텍스트 다음에 올 확률 높은 토큰 예측"

## 대표 모델
| 회사 | 모델 |
|------|------|
| OpenAI | GPT-4, GPT-4o |
| Anthropic | Claude 3.5 |
| Google | Gemini |
| Meta | Llama 3 |
| 네이버 | HyperCLOVA X |

## 학습 단계
1. **Pre-training**: 인터넷 텍스트 대량 학습 (다음 단어 예측)
2. **Fine-tuning**: 목적에 맞게 조정 (SFT, RLHF 등)

## 특징
- 스케일링 법칙: 파라미터 늘리면 성능 올라감
- 창발 능력: 일정 규모 넘으면 갑자기 새로운 능력 등장
- 범용성: 하나의 모델로 번역, 요약, 코딩, 추론 등 다양한 작업 가능
