cp -a 로 테스트 할 항공사에 맞게 이름을 변경해 복사 
``` ubuntu
eborder@kor-iapp-prod-1:~/xaAMR/kor/outbound/NO_HTH$ cp -a 1A1.AC.SND.org AQ1.AQ.SND.org
```

```
eborder@kor-iapp-prod-1:~/xaAMR/kor/outbound/NO_HTH$ vi AQ1.AQ.SND.org




:%s/1A1/AQ/g  # 전체 파일에서 1A1를 찾아 AQ로 모두 치환
```

```
cp AQ1.AQ.SND.org AQ1.AQ.`date +%Y%m%d`.SND; sed -i "s/date/$(date +%y%m%d)/g" AQ1.AQ.`date +%Y%m%d`.SND
파일명에 날짜 붙힌후 
mv AQ1.AQ.20251114.SND ../AQ1
테스트 폴더로 이동
```
