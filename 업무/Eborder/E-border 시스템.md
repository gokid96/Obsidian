# eBorder 시스템 요약

## 개요
출입국 사전정보 시스템으로, 항공사로부터 승객 정보를 수집하여 법무부(CIQ)에 전달하는 시스템.

## 시스템 구성
```
┌──────────────────────────────────────────────────────────────┐
│                      eBorder 시스템                           │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  [PNR] 예약 정보 수집                                          │
│   ├─ Push: 항공사 → PNRGOV/CUSMAN → 파싱 → CIQ                  │
│   └─ Pull: 스케줄 수집 → 항공사 조회 → 파싱 → CIQ                 │
│                                                              │
│  [IAPI] 실시간 출입국 정보                                      │
│   └─ 항공사 ←→ MATIP/MQ ←→ Handler ←→ CIQ                      │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 주요 구성요소

| 구분 | 설명 | 데이터 |
|------|------|--------|
| [[PNR 개요\|PNR]] | 승객 예약 정보 | 예약, 결제, 여정 이력 |
| [[01.IAPI 개요\|IAPI]] | 실시간 출입국 정보 | 여권, 비자, 신원 정보 |

## 기술 스택
- **언어**: Ruby
- **메시지 큐**: Redis, IBM MQ
- **데이터베이스**: MySQL
- **프로토콜**: MATIP, EDIFACT
