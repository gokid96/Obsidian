
## 수신 메시지 (항공사 → E-border)

### PAXLST
승객 정보 메시지 (Passenger List)
```
UNA:+.? '
UNB+UNOA:4+1ASEL01+KRSEL01+231215:1200+12345'
UNH+1+PAXLST:D:05B:UN:IATA'
BGM+745'
TDT+20+OZ123+++OZ'
...
UNT+15+1'
UNZ+1+12345'
```

### USM Reply
USM(Unsolicited Message)에 대한 응답

- BGM+312로 식별
- RFF+TN으로 원본 USM 매칭

## 송신 메시지 (E-border → 항공사)

### CUSRES
PAXLST에 대한 정상 응답 (Customs Response)
```
UNA:+.? '
UNB+UNOA:4+KRSEL01+1ASEL01+231215:1201+12346'
UNH+1+CUSRES:D:05B:UN:IATA'
BGM+962'
...
UNT+10+1'
UNZ+1+12346'
```

### Reject1 (즉시 거부)

| WhyCode | 사유 |
|---------|------|
| 1D | 메시지 크기 초과 |
| 1J | 미등록 항공사 코드 |
| 1K | Layer5 주소 불일치 |

### Reject2 (Timeout 거부)

| WhyCode | 사유 |
|---------|------|
| 1F | 응답 시간 초과 (RequestTimeout) |

### USM (Unsolicited Message)
정부 → 항공사 요청 메시지

- 특정 승객에 대한 추가 정보 요청
- TransactionID로 응답 매칭

## 메시지 카운트 (HandlerStatus)

| 필드 | 설명 |
|------|------|
| PAXLSTMessageCount | 수신 PAXLST 수 |
| CUSRESMessageCount | 송신 CUSRES 수 |
| Reject1MessageCount | Reject1 수 |
| Reject2MessageCount | Reject2 (Timeout) 수 |
| UnsoliciteMessageCount | USM 송신 수 |
| UnsoliciteReplyMessageCount | USM Reply 수신 수 |
