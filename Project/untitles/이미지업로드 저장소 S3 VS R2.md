## S3 vs Cloudflare R2 트레이드오프

| 항목          | S3                        | R2                    |
| ----------- | ------------------------- | --------------------- |
| **무료 티어**   | 5GB, 12개월 한정              | 10GB, 기간 제한 없음        |
| **전송 비용**   | 있음 (GB당 ~$0.09)           | **없음**                |
| **저장 비용**   | ~$0.025/GB                | ~$0.015/GB            |
| **리전 선택**   | 가능 (서울 등)                 | 불가 (자동 분산)            |
| **레이턴시**    | 서울 리전 → 한국 유저에 빠름         | 글로벌 분산, 일관적           |
| **AWS 연동**  | Lambda, CloudFront 등 네이티브 | 별도 설정 필요              |
| **S3 호환성**  | 원본                        | 99% 호환 (일부 기능 미지원)    |
| **CDN**     | CloudFront 별도 설정/비용       | Cloudflare CDN 자동, 무료 |
| **커스텀 도메인** | CloudFront 필요             | 간단 설정                 |
## 현재 아키텍처 기준 

**R2경우**
- 이미지 트래픽 많을 예정
- 비용 예측 가능하게 하고 싶음
- 이미 Cloudflare DNS 사용 중 ← 해당됨

**S3경우**
- Lambda 트리거로 이미지 리사이징 등 처리 필요
- AWS 내 다른 서비스와 긴밀한 연동 필요
- 한국 유저 대상, 최소 레이턴시 중요

**결론**: 단순 이미지 저장/서빙이면 **R2**, AWS 생태계 활용할 거면 **S3**
