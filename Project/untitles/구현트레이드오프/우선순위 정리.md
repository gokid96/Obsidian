## 구현된 부분

|항목|설명|
|---|---|
|**낙관적 락**|`@Version` + 409 응답 + 프론트 충돌 모달|
|**세션 인증**|Spring Security + 쿠키 기반|
|**권한 체계**|OWNER > ADMIN > MEMBER > VIEWER 레벨 시스템|
|**트리 구조**|폴더 + 노트 한 번에 조회|
|**자동 저장**|debounce 1초 + 저장 직렬화|
|**API 일관성**|백엔드-프론트 엔드포인트 완벽 매칭|
|**CI/CD 배포**|GitHub Actions + Docker + EC2 자동 배포|
|**SSL 인증서**|Caddy (Let's Encrypt 자동 발급/갱신)|
|**도메인 연결**|untitles.net (Vercel) + api.untitles.net (EC2)|
