

#### EC2 인스턴스 초기 사양

| EC2/RDS | t3.small | db.t4g.micro |
| ------- | -------- | ------------ |
| 메모리     | 2GB      | 1 GB         |
| CPU     | 2vCPU    | 2 vCPU       |


```
ubuntu@ip-172-31-60-0:~/.docker$ free -h
               total        used        free      shared  buff/cache   available
Mem:           1.9Gi       879Mi       141Mi       2.9Mi       1.1Gi       1.0Gi
Swap:             0B          0B          0B
```

- 전체 2GB 중 available 1GB (Spring Boot + Docker + Caddy가 기본으로 먹는 양)
- 약 50% 여유

사용자 늘어나면 서버 터짐 , 트레이드 오프 

- **지금**: JVM 메모리 튜닝 + 불필요 프로세스 정리 + RDS idle 커넥션 정리 t3.medium(4GB)으로 업그레이드 + 스왑 추가 후 

- **트래픽 늘면**: 세션을 Redis로 전환 → Auto Scaling 구성 또는 아직 Redis 사용하지않고 JWT 토큰 방식 로그인으로 전환후 Auto Scaling

- **더 커지면**: RDS도 db.t4g.small(2GB)로 업그레이드
