### 게시물 이미지 삭제가 어려운 이유

```html
<!-- 게시물 content (HTML) -->
<p>안녕하세요</p>
<img src="https://r2.../posts/uuid1.jpg">
<p>중간 텍스트</p>
<img src="https://r2.../posts/uuid2.jpg">
```

- 수정 시: 기존 content와 새 content를 비교해서 **삭제된 이미지 URL**을 찾아야 함
- 삭제 시: content에서 모든 이미지 URL을 **파싱**해서 삭제해야 함

→ HTML 파싱 + diff 로직 필요 → 복잡도 높음
