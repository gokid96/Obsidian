### EC2 (Elastic Compute Cloud) 
- 클라우드 가상 서버 
- 용도별 인스턴스 타입 선택 (t3.micro, t3.large 등) 
### EBS (Elastic Block Store) 
- EC2 전용 하드디스크 
- 인스턴스 1개에만 연결 
- 스냅샷으로 백업 가능 
### S3 (Simple Storage Service) 
- 무제한 클라우드 저장소 
- 어디서든 HTTP API로 접근 
- EBS보다 저렴, 백업/보관용 
### 보안 그룹 (Security Group) 
 - 인스턴스 레벨 방화벽 
 - 인바운드/아웃바운드 규칙 설정 
 - 포트별, IP별 접근 제어 
### 기타 EC2 리소스 
- **로드 밸런서**: 트래픽을 여러 서버에 분산
- **탄력적 IP**: 재시작해도 안 바뀌는 고정 IP 
- **Auto Scaling**: 트래픽에 따라 자동 서버 증감 
- **키 페어**: SSH 접속용 .pem 파일 
- **스냅샷**: 디스크 백업 (Git commit 개념)
