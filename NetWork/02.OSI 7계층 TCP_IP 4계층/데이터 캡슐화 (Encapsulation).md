```
송신 측 (위 → 아래)

┌─────────────────────────────────────────┐

│ Application  │ Data                     │

├──────────────┼──────────────────────────┤

│ Transport    │ TCP Header + Data        │  ← Segment

├──────────────┼──────────────────────────┤

│ Network      │ IP Header + Segment      │  ← Packet

├──────────────┼──────────────────────────────┤

│ Data Link    │ Frame Header + Packet + FCS │← Frame

├──────────────┼──────────────────────────────┤

│ Physical     │ 010110101...               │  ← Bits

└──────────────┴──────────────────────────────┘

```


수신 측은 역순으로 역캡슐화 (Decapsulation)
