## 1. **방화벽(Firewall)**

- **목적:** 네트워크 보안, 허용/차단 정책 관리
    
- **역할:**
    - 들어오는 트래픽(외부 → 내부)과 나가는 트래픽(내부 → 외부)을 검사
    - 정책에 따라 허용/차단
        
- **종류:**
    - **하드웨어 방화벽:** 라우터, 전용 장비
    - **소프트웨어 방화벽:** 서버, PC에 설치
        
- **예시:**
    - 특정 IP만 접속 허용
    - 특정 포트(예: 80, 443)만 열어 웹 서비스 가능

## 2. **NAT(Network Address Translation)**

- **목적:** 사설 IP ↔ 공인 IP 변환
    
- **왜 필요한가:**
    - LAN 내부는 사설 IP(192.168.x.x 등)를 쓰지만 인터넷은 공인 IP가 필요
        
- **역할:**
    - 내부 장치들이 인터넷과 통신할 수 있게 변환
    - 외부에서 내부 장치를 바로 접근하지 못하게 보안 기능도 겸함
        
- **종류:**
    - **SNAT(Source NAT):** 내부 IP → 공인 IP 변환 (주로 인터넷 접속 시)
    - **DNAT(Destination NAT):** 외부 요청 → 내부 서버 IP로 포워딩 (포트 포워딩)

---

### 정리하면

- **방화벽** = 트래픽 허용/차단, 보안 관리
- **NAT** = 내부 IP를 외부 인터넷과 연결 가능하게 변환
