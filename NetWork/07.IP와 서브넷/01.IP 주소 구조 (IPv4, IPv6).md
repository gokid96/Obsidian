## IPv4

- 32비트, 4개 옥텟 (8비트씩)
- 표기: 192.168.1.1
- 총 약 43억 개 (2^32)
```
192    .   168    .    1     .    1
11000000  10101000  00000001  00000001
```

### IPv4 클래스 (레거시)

|클래스|범위|네트워크/호스트 비트|기본 서브넷|
|---|---|---|---|
|A|0.0.0.0 ~ 127.255.255.255|8/24|/8|
|B|128.0.0.0 ~ 191.255.255.255|16/16|/16|
|C|192.0.0.0 ~ 223.255.255.255|24/8|/24|
|D|224.0.0.0 ~ 239.255.255.255|멀티캐스트|-|
|E|240.0.0.0 ~ 255.255.255.255|예약|-|

### 사설 IP 대역

|클래스|대역|CIDR|
|---|---|---|
|A|10.0.0.0 ~ 10.255.255.255|10.0.0.0/8|
|B|172.16.0.0 ~ 172.31.255.255|172.16.0.0/12|
|C|192.168.0.0 ~ 192.168.255.255|192.168.0.0/16|

### 특수 IP

|주소|용도|
|---|---|
|127.0.0.1|루프백 (localhost)|
|0.0.0.0|모든 IP (INADDR_ANY)|
|255.255.255.255|브로드캐스트|

## IPv6

- 128비트, 8개 그룹 (16비트씩)
- 표기: 2001:0db8:0000:0000:0000:0000:0000:0001
- 축약: 2001:db8::1

### IPv6 축약 규칙

1. 각 그룹 앞의 0 생략: 0db8 → db8
2. 연속된 0 그룹은 :: 로 한 번만 축약

```
2001:0db8:0000:0000:0000:0000:0000:0001
→ 2001:db8::1
```

### IPv6 주소 유형

|유형|범위|용도|
|---|---|---|
|::1/128|루프백|localhost|
|fe80::/10|링크 로컬|같은 네트워크 내|
|2000::/3|글로벌 유니캐스트|인터넷 라우팅|
