- 인터페이스 DOWN → 로컬 문제
- 게이트웨이 안 됨 → LAN 문제
- 외부 IP 안 됨 → **인터넷 회선/ISP 문제** ← 이게 빠졌어요!
- 도메인 안 됨 → DNS 문제
- 특정 사이트만 안 됨 → 방화벽/해당 사이트 문제
```
		인터넷 안됨 
				↓
		ip addr show → IP 없음? → DHCP 또는 네트워크 케이블 문제 
				↓ 
		IP 있음 ping 게이트웨이 → 실패? → 로컬 네트워크 문제 (스위치/라우터) 
				↓ 
		성공 ping 8.8.8.8 → 실패? → 라우팅/ISP 문제 
				↓
		성공 ping google.com → 실패? → DNS 문제 
				↓ 
		성공 curl google.com → 실패? → 방화벽/프록시 문제
```

## **빠른 체크 스크립트:**
```bash
#!/bin/bash
echo "=== 네트워크 진단 시작 ==="
echo ""
echo "1. IP 주소:"
ip addr show | grep "inet "
echo ""
echo "2. 게이트웨이:"
ip route | grep default
echo ""
echo "3. DNS 서버:"
cat /etc/resolv.conf | grep nameserver
echo ""
echo "4. 게이트웨이 Ping:"
ping -c 2 $(ip route | grep default | awk '{print $3}')
echo ""
echo "5. 외부 DNS Ping:"
ping -c 2 8.8.8.8
echo ""
echo "6. 도메인 DNS 조회:"
nslookup google.com
```
