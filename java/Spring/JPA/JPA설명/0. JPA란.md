## JPA란?

- **Java Persistence API**: 자바 ORM 기술의 표준
- **ORM**: 객체와 관계형 데이터베이스를 자동으로 매핑
- SQL 없이 객체로 데이터베이스 조작


```java
// 기존 JDBC
String sql = "INSERT INTO user VALUES(?,?)";
pstmt.setString(1, user.getName());
// 반복 코드, SQL 의존

// JPA
em.persist(user);  // 끝!
```

**장점**: 생산성 향상, 유지보수 쉬움, DB 변경 용이
