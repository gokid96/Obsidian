## 1. 사업 개요

### 1.1 서비스 명칭

**"주식왕 키즈"** (가칭)

### 1.2 한 줄 소개

초등학생이 한국 주식으로 재미있게 배우고, 수익 달성 시 실제 용돈을 받는 게임형 금융교육 플랫폼

### 1.3 핵심 차별점

- **국내 최초** 아동 전용 한국 주식 모의투자 서비스
- 학습 성과를 **실제 용돈**으로 보상하는 동기부여 시스템
- 게임처럼 즐겁게 배우는 금융교육

---

## 2. 시장 분석

### 2.1 시장 기회

**타겟 시장 규모**

- 국내 초등학생 인구: 약 270만 명 (2025년 기준)
- 키즈 시장 규모: 40조 원 (2017년 기준, 지속 성장)
- 미성년자 주식 계좌 보유자: 75만 명 (2023년 말 기준, 8배 증가)

**시장 트렌드**

- 조기 금융교육에 대한 부모 관심 급증
- 미성년자 주식투자 활성화 (비대면 계좌 개설 가능)
- 게이미피케이션 교육 콘텐츠 선호도 증가

### 2.2 경쟁사 분석

|서비스|타겟|주요 기능|한계점|
|---|---|---|---|
|**토스 모의투자**|만 7-18세|미국 주식 모의투자|미국 주식만, 교육 콘텐츠 부족|
|**하나은행 아이부자**|만 14세 미만|용돈 관리, 주식 체험|실제 거래만 가능, 모의투자 없음|
|**퍼핀**|초등학생|용돈 관리, 금융교육|주식투자 기능 없음|
|**증권사 모의투자**|성인 위주|실전 모의투자|어린이에게 너무 복잡한 UI|

**우리의 포지셔닝** → 한국 주식 + 아동 맞춤 + 용돈 연동 = **유일무이한 서비스**

## 3. 서비스 상세 기획

### 3.1 타겟 고객

**Primary (1차 타겟)**

- 초등학교 3~6학년 (만 9~12세)
- 경제/금융에 관심 있는 학생
- 용돈을 받고 있는 학생

**Secondary (2차 타겟)**

- 자녀에게 경제교육을 시키고 싶은 30-40대 부모
- 합리적 소비 습관을 가르치고 싶은 부모
- IT 친화적인 부모

### 3.2 핵심 기능

#### A. 모의투자 시스템

**1) 초기 자금**

- 가입 시 가상 자금 100만 원 지급
- 매일 출석 시 1만 원 추가 지급
- 미션 완수 시 추가 자금 획득

**2) 거래 가능 종목**

- 코스피/코스닥 상장 주식 중 아이들이 아는 기업 우선
    - 삼성전자, LG전자, 현대차, 카카오, 네이버, 엔씨소프트 등
    - 게임사, 엔터테인먼트사, 식품회사 등
- ETF (처음 배우기 좋은 상품)

**3) 실시간 데이터 연동**

- 실제 주식 시세 반영 (15분 지연)
- 장 개장 시간에만 거래 가능 (실제와 동일)
- 호가 시스템 적용

#### B. 게이미피케이션 요소

**1) 레벨 시스템**

```
🌱 Lv.1-5: 주식 새싹 (초보 투자자)
🌿 Lv.6-10: 투자 모종 (배우는 중)
🌳 Lv.11-15: 투자 나무 (경험자)
🏆 Lv.16-20: 투자 전문가 (고수)
👑 Lv.21+: 주식왕 (마스터)
```

**2) 뱃지 수집**

- "첫 주식 매수" 뱃지
- "배당주 수집가" 뱃지
- "장기 투자자" 뱃지 (30일 이상 보유)
- "뉴스 마스터" 뱃지 (뉴스 100개 읽기)
- "수익 달성" 뱃지 (5%, 10%, 20% 수익)

**3) 일일/주간 미션**

```
📅 일일 미션 (매일 초기화)
✅ 로그인하기 → 가상머니 +10,000원
✅ 오늘의 경제 뉴스 읽기 → 가상머니 +5,000원
✅ 1개 종목 분석하기 → 가상머니 +10,000원

📊 주간 미션 (주간 초기화)
✅ 5일 연속 접속 → 가상머니 +50,000원
✅ 3개 종목 매수하기 → 가상머니 +30,000원
✅ 금융 퀴즈 10개 맞히기 → 가상머니 +40,000원
```

#### C. 수익 연동 용돈 시스템 ⭐ (핵심 차별화)

**1) 작동 방식**

```
Step 1: 부모가 목표 설정
- 목표 수익률: 10% 달성 시
- 보상 금액: 5,000원
- 월 최대 한도: 20,000원

Step 2: 아이가 목표 달성
- 포트폴리오 수익률 10% 돌파
- "용돈 요청" 버튼 활성화

Step 3: 용돈 요청
- 아이가 요청 버튼 클릭
- 부모 앱에 알림 발송
- "민수가 수익률 10% 달성! 축하 용돈 5,000원 보내기"

Step 4: 부모 승인
- 부모가 앱에서 승인
- 아이 계좌로 실제 용돈 송금
- 축하 메시지 + 다음 목표 제시
```

**2) 용돈 요청 트리거 (부모가 선택)**

|트리거 유형|예시|추천 보상|
|---|---|---|
|수익률 달성|+5%, +10%, +20%|3,000~10,000원|
|연속 수익일|5일, 10일 연속 플러스|5,000~8,000원|
|투자 미션|배당주 5개 매수|3,000원|
|학습 완료|금융교육 10강 수료|5,000원|
|장기 투자|30일 이상 보유|10,000원|
|포트폴리오 분산|5개 이상 종목 보유|4,000원|

**3) 부모 대시보드**

- 자녀 포트폴리오 실시간 조회
- 투자 성향 분석 리포트
- 용돈 지급 내역
- 월간 학습 리포트
- 목표 달성률 그래프

**4) 안전장치**

- 월 최대 용돈 한도 설정
- 용돈 요청 가능 횟수 제한 (월 4회 등)
- 손실 시에도 학습 보상 (과정 중시)
- "투자 일기" 작성 시 추가 보상

#### D. 교육 콘텐츠

**1) 연령별 맞춤 커리큘럼**

**[초급] 초등 3-4학년**

- 주식이란 무엇인가?
- 회사와 주주의 관계
- 주식을 사고파는 방법
- 좋아하는 회사 주식 찾기

**[중급] 초등 5-6학년**

- 주가가 오르고 내리는 이유
- 배당이란?
- 뉴스 읽고 이해하기
- 분산 투자의 중요성

**[고급] 중학생 이상**

- 재무제표 기초
- 산업 분석
- 장기 투자 vs 단기 투자
- ETF와 펀드

**2) 콘텐츠 형식**

- 3분 애니메이션 영상
- 웹툰 형식 학습 만화
- 인터랙티브 퀴즈
- 실제 뉴스 쉽게 풀어쓰기
- "오늘의 기업 이야기"

#### E. 커뮤니티 기능

**1) 안전한 키즈 커뮤니티**

- 실명 비공개 (닉네임만)
- 욕설/비속어 자동 필터링
- 신고 시스템
- 부모 모니터링 가능

**2) 소셜 기능**

- 친구와 수익률 비교
- 반별/학교별 리그
- 이번 주 TOP 투자자
- 포트폴리오 공유 (선택적)

**3) 챌린지**

- "한 달 장기투자 챌린지"
- "배당주 수집 챌린지"
- "경제뉴스 읽기 챌린지"
