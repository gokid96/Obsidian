## 포트폴리오 스토리 
``` 
프로젝트 소개: "숨고와 당근마켓에서 영감을 받아 만든 위치 기반 PT 트레이너 매칭 플랫폼입니다. 프리랜서 트레이너와 회원을 연결하고, 스케줄 관리 및 운동 기록 관리 기능을 제공합니다." 
문제 정의: - 프리랜서 트레이너는 회원 관리가 어렵다 
- 회원은 내 근처 좋은 트레이너 찾기 어렵다 
- 기존 플랫폼은 특정 헬스장에 종속적이다 
해결 방법: 
- 위치 기반 트레이너 검색 
- 실시간 채팅으로 직접 소통 
- 트레이너용 회원 관리 도구 제공 
기술적 도전: 
1. 위치 기반 검색 최적화
2. 스케줄 충돌 방지 (동시성) 
3. 실시간 채팅 구현 
4. 리뷰 집계 성능 개선 
```
