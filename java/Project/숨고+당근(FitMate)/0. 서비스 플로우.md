### 1. 회원 시나리오
```
1. 위치 기반으로 근처 트레이너 검색
2. 트레이너 프로필 확인 (경력, 리뷰, 전문분야)
3. "PT 신청" 버튼
4. 채팅으로 일정/장소 협의
   - "XX 헬스장에서 월수금 7시 가능하세요?"
   - "1회당 5만원입니다"
5. 협의 완료되면 정기 일정 등록
6. 결제는 직거래 or 서비스 내 결제
```

### 2. 트레이너 시나리오
```
1. 프로필 등록 (경력, 자격증, 전문분야)
2. 활동 가능 헬스장/지역 등록
   - "강남구 XX헬스장, YY헬스장에서 수업 가능"
3. 회원 신청 알림 받음
4. 채팅으로 상담
5. 스케줄 관리 (캘린더)
6. 회원 운동 기록 관리
```

## 핵심 기능 재정의

### 필수 기능 (MVP)

#### 1. 트레이너 프로필
```
- 기본 정보 (이름, 사진, 소개)
- 전문 분야 (체중 감량, 재활, 보디빌딩 등)
- 경력 (년수, 자격증)
- 활동 지역/헬스장
- 가격 정보
- 리뷰/평점
```

#### 2. 검색 & 필터링
```
- 위치 기반 검색 (내 위치 3km 이내)
- 카테고리 필터 (전문분야)
- 가격대 필터
- 평점 순 정렬
```

#### 3. 매칭 & 채팅
```
- PT 신청하기
- 실시간 채팅
- 일정 조율
```

#### 4. 스케줄 관리 (트레이너용)
```
- 주간 캘린더
- 회원별 예약 현황
- 일정 수정/취소
```

#### 5. 운동 기록 관리 (트레이너용)
```
- 회원별 운동 기록
- 체중/신체 사이즈 기록
- 진행 상황 메모
```
