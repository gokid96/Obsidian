## 📋 1. 초보 투자자 시나리오

### 1단계: 회원가입 & 레벨 테스트

```
[회원가입]
1. 이메일/소셜 로그인
2. 기본 정보 입력
   - 닉네임, 관심 분야
3. 투자 경험 선택
   □ 완전 초보 (주식이 뭔지 모름)
   □ 입문자 (용어는 알지만 매매 경험 없음)
   □ 초급자 (몇 번 해봤지만 손실)
4. 간단한 레벨 테스트 (10문제)
   - "시가총액이란?"
   - "PER의 의미는?"
   - 결과에 따라 추천 학습 경로 제시
5. 가상 투자금 1,000만원 지급
```

### 2단계: 튜토리얼 & 첫 주식 매수

```
[튜토리얼 모드]
1. 가이드: "처음이시군요! 함께 시작해봐요"
2. 주식 시장 둘러보기
   - 인기 종목 TOP 10 보기
   - 각 종목 간단 설명
   
3. 첫 주식 고르기 (추천 3개)
   예시:
   ┌─────────────────────────┐
   │ 🏢 삼성전자              │
   │ 현재가: 70,000원         │
   │ 💡 초보자 추천: ⭐⭐⭐⭐⭐ │
   │                         │
   │ 왜 추천?                │
   │ - 안정적인 대기업       │
   │ - 거래량 많음 (유동성)  │
   │ - 초보자가 배우기 좋음  │
   │                         │
   │ [자세히 보기] [매수하기]│
   └─────────────────────────┘

4. 매수 체험
   - "몇 주 살까요?" 입력
   - "총 금액: 700,000원"
   - [확인] 클릭
   
5. 매수 완료 축하 🎉
   - "첫 투자를 축하합니다!"
   - 배지 획득: 🏆 첫 주식 투자자
   - 투자 일지 자동 생성
```

### 3단계: 학습 & 실습 사이클

```
[일일 미션]
오늘의 학습:
┌─────────────────────────┐
│ 📚 오늘의 주제: PER      │
│                         │
│ 1. 개념 학습 (5분)      │
│ 2. 퀴즈 (3문제)         │
│ 3. 실습: PER 낮은        │
│    주식 찾기            │
│                         │
│ 완료 시 보상:           │
│ 💰 가상 투자금 +10만원  │
│ ⭐ 경험치 +100          │
└─────────────────────────┘

[학습 흐름]
1. 영상/텍스트로 개념 학습
   - PER이란? (Price Earning Ratio)
   - 주가 ÷ 주당순이익
   - 낮을수록 저평가
   
2. 이해도 퀴즈
   Q1. PER 15와 PER 30, 어느 것이 저평가?
   → 정답: PER 15
   
3. 실전 적용
   "삼성전자 PER 확인해보기"
   - 화면에 PER 강조 표시
   - "현재 PER: 12.5"
   - "업종 평균: 15.0"
   - 💡 판단: 상대적 저평가
   
4. 미션: PER 10 이하 종목 찾아서 매수하기
```

## 📋 2. 관리자/시스템 시나리오

### 자동화 프로세스

```
[매일 자정 - Scheduler]

2. 랭킹 업데이트
3. 일일 리포트 생성
   - 오늘의 수익 TOP 3
   - 가장 많이 거래된 종목

[장 시작 전 - 오전 8시]
1. 주요 뉴스 크롤링
2. 주요 지수 업데이트
3. 사용자에게 알림
   "오늘 코스피 예상: 상승세"

[장 중 - 9시~3시 30분]
1. 실시간 주가 크롤링 (1분마다)
2. Redis 캐싱
3. WebSocket으로 실시간 푸시

[장 마감 후 - 오후 4시]
1. 오늘의 거래 정산
2. 포트폴리오 수익률 계산
```
