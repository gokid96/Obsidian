## 변수란?
값을 저장할 수 있는 메모리 공간. 하나의 값만 저장 가능.

## 변수 선언 규칙
- 첫 글자: 문자, $, _ (숫자 불가)
- 대소문자 구분
- 예약어 사용 불가 (int, class, public 등)
- 카멜케이스 권장: myName, totalCount

---

## 기본 타입 (Primitive Type)

스택 메모리에 실제 값 저장

| 구분 | 타입 | 크기 | 범위 | 기본값 |
|------|------|------|------|--------|
| 정수 | byte | 1byte | -128 ~ 127 | 0 |
| | short | 2byte | -32,768 ~ 32,767 | 0 |
| | int | 4byte | 약 -21억 ~ 21억 | 0 |
| | long | 8byte | 매우 큼 | 0L |
| 실수 | float | 4byte | 소수점 약 7자리 | 0.0f |
| | double | 8byte | 소수점 약 15자리 | 0.0 |
| 문자 | char | 2byte | 0 ~ 65,535 | '\u0000' |
| 논리 | boolean | 1bit | true / false | false |

---

## 참조 타입 (Reference Type)

힙 메모리의 주소값 저장

- String
- 배열
- 클래스
- 인터페이스
- 열거형 (enum)

### 기본 타입 vs 참조 타입
| 구분 | 기본 타입 | 참조 타입 |
|------|-----------|-----------|
| 저장 값 | 실제 값 | 메모리 주소 |
| 메모리 | 스택 | 스택(주소) + 힙(실제값) |
| null | 불가능 | 가능 |
| 비교 | == (값 비교) | == (주소), equals() (값) |

---

## 타입 변환

### 자동 형변환 (묵시적)
작은 타입 → 큰 타입 (데이터 손실 없음)

byte → short → int → long → float → double
         char ↗

### 강제 형변환 (명시적)
큰 타입 → 작은 타입 (데이터 손실 가능)

(타입) 값

## var 키워드 (Java 10+)

컴파일러가 타입 추론

### 사용 가능
- 지역 변수
- for문 변수

### 사용 불가
- 멤버 변수 (필드)
- 메서드 매개변수
- 반환 타입
- null 초기화
