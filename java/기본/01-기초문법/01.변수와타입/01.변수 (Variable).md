## 변수란?

- 데이터를 저장하는 **메모리 공간**의 이름
- 프로그램 실행 중 값을 저장하고 변경할 수 있음

```java
int age = 25;           // 변수 선언과 초기화
String name = "홍길동";  // 문자열 변수
```

---

## 변수 선언과 초기화

### 선언 (Declaration)

```java
int number;        // 선언만
String text;       // 선언만
```

### 초기화 (Initialization)

```java
number = 10;       // 초기화
text = "Hello";    // 초기화
```

### 선언과 동시에 초기화

```java
int age = 25;
double pi = 3.14;
boolean isActive = true;
```

## 변수 명명 규칙

### 필수 규칙 (어기면 컴파일 에러)

|규칙|올바른 예|잘못된 예|
|---|---|---|
|첫 글자는 문자, $, _|`name`, `$price`, `_count`|`1stPlace`, `@email`|
|대소문자 구분|`age`와 `Age`는 다름|-|
|예약어 사용 불가|`myClass`|`class`, `int`, `for`|
|공백 불가|`firstName`|`first name`|

### 권장 규칙 (관례)

- **카멜 케이스**: `firstName`, `totalCount`, `isEnabled`
- **의미 있는 이름**: `a` ❌ → `age` ✅
- **상수는 대문자**: `MAX_SIZE`, `PI`

## 변수의 종류

### 1. 지역 변수 (Local Variable)

- 메서드 내부에서 선언
- 메서드 종료 시 소멸
- **반드시 초기화 필요**

```java
void calculate() {
    int result = 0;  // 지역 변수
    result = 10 + 20;
    System.out.println(result);
}  // 여기서 result 소멸
```

### 2. 인스턴스 변수 (Instance Variable)

- 클래스 내부, 메서드 외부에서 선언
- 객체마다 별도로 존재
- 자동 초기화됨

```java
class Person {
    String name;  // 인스턴스 변수 (기본값: null)
    int age;      // 인스턴스 변수 (기본값: 0)
    
    void introduce() {
        System.out.println(name + ", " + age);
    }
}
```

### 3. 클래스 변수 (Static Variable)

- `static` 키워드 사용
- 모든 인스턴스가 **공유**
- 클래스 로딩 시 생성

```java
class Counter {
    static int totalCount = 0;  // 클래스 변수
    int myCount;                // 인스턴스 변수
    
    Counter() {
        totalCount++;  // 모든 객체가 공유
        myCount++;     // 각 객체별로 별도
    }
}
```

### 변수 종류 비교

|구분|지역 변수|인스턴스 변수|클래스 변수|
|---|---|---|---|
|선언 위치|메서드 내부|클래스 내부|클래스 내부 (static)|
|생성 시점|메서드 호출|객체 생성|클래스 로딩|
|소멸 시점|메서드 종료|객체 소멸|프로그램 종료|
|기본값|없음 (초기화 필수)|타입별 기본값|타입별 기본값|
|공유 범위|해당 메서드|해당 객체|모든 객체|

## 상수 (Constant)

- `final` 키워드 사용
- **한 번 초기화하면 변경 불가**
- 대문자 + 언더스코어 권장

```java
final double PI = 3.14159;
final int MAX_SIZE = 100;
final String COMPANY_NAME = "ABC Corp";

// PI = 3.14;  // 컴파일 에러!
```

## var 키워드 (Java 10+)

- 타입 추론: 컴파일러가 자동으로 타입 결정
- **지역 변수에서만 사용 가능**
- 초기화 필수

```java
var num = 100;              // int로 추론
var name = "홍길동";         // String으로 추론
var list = new ArrayList<String>();  // ArrayList<String>으로 추론

// var data;  // 에러! 초기화 필수
```

### var 사용 시 주의

```java
var result = 10 / 3;    // int로 추론 → 결과: 3
var result2 = 10 / 3.0; // double로 추론 → 결과: 3.333...
```

## 변수의 스코프 (Scope)

- 변수를 사용할 수 있는 **유효 범위**

```java
public class ScopeExample {
    int instanceVar = 1;  // 클래스 전체에서 사용 가능
    
    void method() {
        int localVar = 2;  // 이 메서드에서만 사용 가능
        
        if (true) {
            int blockVar = 3;  // 이 블록에서만 사용 가능
        }
        // blockVar 사용 불가
    }
    // localVar 사용 불가
}
```

---

> [!tip] 핵심 정리
> 
> - 변수는 데이터를 저장하는 메모리 공간
> - 지역/인스턴스/클래스 변수 3종류
> - 카멜 케이스로 의미 있는 이름 사용
> - 상수는 `final` + 대문자
> - `var`는 지역 변수에서만 (Java 10+)
