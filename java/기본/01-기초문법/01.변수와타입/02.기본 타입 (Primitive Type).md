## 개요

- Java가 제공하는 **8가지 기본 데이터 타입**
- 실제 값을 **Stack 메모리**에 직접 저장
- 소문자로 시작 (`int`, `double` 등)

---

## 정수형 (Integer Types)

### 타입별 크기와 범위

| 타입      | 크기            | 범위                             | 기본값 |
| ------- | ------------- | ------------------------------ | --- |
| `byte`  | 1byte (8bit)  | -128 ~ 127                     | 0   |
| `short` | 2byte (16bit) | -32,768 ~ 32,767               | 0   |
| `int`   | 4byte (32bit) | -2,147,483,648 ~ 2,147,483,647 | 0   |
| `long`  | 8byte (64bit) | 약 -922경 ~ 922경                 | 0L  |

### 사용 예시

```java
byte b = 127;           // 작은 범위 정수
short s = 32000;        // 중간 범위 정수
int num = 100;          // 일반적인 정수 (기본)
long bigNum = 10000000000L;  // 큰 정수 (L 필수!)
```

### 진법 표현

```java
int decimal = 100;      // 10진수
int binary = 0b1100100; // 2진수 (0b 접두사) → 100
int octal = 0144;       // 8진수 (0 접두사) → 100
int hex = 0x64;         // 16진수 (0x 접두사) → 100
```

### 언더스코어로 가독성 향상 (Java 7+)

```java
int million = 1_000_000;        // 1000000
long credit = 1_234_567_890L;   // 가독성 향상
```

> [!warning] 주의: 오버플로우
> 
> ```java
> int max = Integer.MAX_VALUE;  // 2,147,483,647
> int overflow = max + 1;       // -2,147,483,648 (오버플로우!)
> ```

## 실수형 (Floating-Point Types)

### 타입별 크기와 정밀도

|타입|크기|정밀도|기본값|
|---|---|---|---|
|`float`|4byte|소수점 약 7자리|0.0F|
|`double`|8byte|소수점 약 15자리|0.0|

### 사용 예시

```java
float f = 3.14F;        // F 접미사 필수!
double d = 3.141592;    // 기본 실수 타입
double sci = 1.5e3;     // 과학적 표기법 → 1500.0
double small = 2.5e-3;  // → 0.0025
```

### float vs double

```java
float f = 0.1F;
double d = 0.1;

System.out.println(f);  // 0.1
System.out.println(d);  // 0.1

// 정밀도 차이
float f2 = 0.12345678901234567890F;
double d2 = 0.12345678901234567890;
System.out.println(f2);  // 0.12345679 (7자리까지)
System.out.println(d2);  // 0.12345678901234568 (15자리까지)
```

> [!warning] 주의: 부동소수점 오차
> 
> ```java
> double result = 0.1 + 0.2;
> System.out.println(result);  // 0.30000000000000004
> System.out.println(result == 0.3);  // false!
> 
> // 해결: BigDecimal 사용
> BigDecimal a = new BigDecimal("0.1");
> BigDecimal b = new BigDecimal("0.2");
> System.out.println(a.add(b));  // 0.3
> ```

## 문자형 (Character Type)

### char 타입

|타입|크기|범위|기본값|
|---|---|---|---|
|`char`|2byte (16bit)|0 ~ 65,535 (유니코드)|'\u0000'|

### 사용 예시

```java
char ch1 = 'A';           // 문자 리터럴
char ch2 = 65;            // 숫자로 표현 → 'A'
char ch3 = '\uAC00';      // 유니코드 → '가'
char ch4 = '\n';          // 이스케이프 문자 (줄바꿈)
```

### 이스케이프 문자

|문자|의미|
|---|---|
|`\n`|줄바꿈 (new line)|
|`\t`|탭 (tab)|
|`\\`|역슬래시|
|`\'`|작은따옴표|
|`\"`|큰따옴표|

### char와 int 변환

```java
char ch = 'A';
int ascii = ch;           // 65 (자동 변환)
char ch2 = (char) 66;     // 'B' (강제 변환)

// 문자 연산
char next = (char) ('A' + 1);  // 'B'
```

## 논리형 (Boolean Type)

### boolean 타입

|타입|크기|값|기본값|
|---|---|---|---|
|`boolean`|1bit (JVM 구현에 따라 다름)|true / false|false|

### 사용 예시

```java
boolean isActive = true;
boolean isFinished = false;

// 조건식 결과
boolean result = (10 > 5);  // true
boolean check = (10 == 5);  // false
```

### 주의: Java에서 boolean

```java
// C/C++와 다르게 숫자를 boolean으로 사용 불가
// if (1) { }  // 컴파일 에러!
if (true) { }  // OK

// 0, 1로 변환하려면 명시적으로
int num = isActive ? 1 : 0;
```

## 기본값 정리

```java
class DefaultValues {
    byte b;      // 0
    short s;     // 0
    int i;       // 0
    long l;      // 0L
    float f;     // 0.0F
    double d;    // 0.0
    char c;      // '\u0000' (공백)
    boolean bo;  // false
}
```

> [!caution] 지역 변수는 기본값 없음
> 
> ```java
> void method() {
>     int local;
>     System.out.println(local);  // 컴파일 에러!
> }
> ```

## 리터럴 (Literal)

- 소스 코드에서 **직접 표현한 값**

```java
// 정수 리터럴
int a = 100;       // 10진수
int b = 0b1010;    // 2진수
int c = 0144;      // 8진수
int d = 0x64;      // 16진수

// 실수 리터럴
double e = 3.14;   // 일반 표기
double f = 3.14e2; // 지수 표기

// 문자 리터럴
char g = 'A';
char h = '\n';

// 논리 리터럴
boolean i = true;
boolean j = false;
```

---

> [!tip] 핵심 정리
> 
> - 정수형: `byte` < `short` < `int` < `long` (기본: int)
> - 실수형: `float` < `double` (기본: double)
> - `long`은 `L`, `float`은 `F` 접미사 필수
> - `char`는 유니코드 (2byte)
> - `boolean`은 true/false만 가능
