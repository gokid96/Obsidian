## 개요

- 수학적 계산을 수행하는 연산자
- 사칙연산 + 나머지 연산

---

## 기본 산술 연산자

|연산자|의미|예시|결과|
|---|---|---|---|
|`+`|덧셈|`5 + 3`|8|
|`-`|뺄셈|`5 - 3`|2|
|`*`|곱셈|`5 * 3`|15|
|`/`|나눗셈|`5 / 3`|1|
|`%`|나머지|`5 % 3`|2|

```java
int a = 10, b = 3;

System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3 (정수 나눗셈)
System.out.println(a % b);  // 1 (나머지)
```

---

## 나눗셈 주의사항

### 정수 나눗셈

```java
int a = 10, b = 4;
int result = a / b;  // 2 (소수점 버림)

// 실수 결과를 원한다면
double result2 = (double) a / b;  // 2.5
double result3 = a / (double) b;  // 2.5
```

### 0으로 나누기

```java
// 정수: ArithmeticException 발생
int a = 10 / 0;  // 예외!

// 실수: Infinity 또는 NaN
double b = 10.0 / 0;   // Infinity
double c = 0.0 / 0;    // NaN (Not a Number)
```

## 나머지 연산자 (%)

### 기본 사용

```java
System.out.println(10 % 3);   // 1
System.out.println(10 % 5);   // 0
System.out.println(10 % 7);   // 3
```

### 활용 예시

```java
// 짝수/홀수 판별
int num = 7;
if (num % 2 == 0) {
    System.out.println("짝수");
} else {
    System.out.println("홀수");
}

// 배수 판별
if (num % 3 == 0) {
    System.out.println("3의 배수");
}

// 순환 (0, 1, 2, 0, 1, 2, ...)
for (int i = 0; i < 10; i++) {
    System.out.println(i % 3);
}
```

### 음수의 나머지

```java
System.out.println(10 % 3);    // 1
System.out.println(-10 % 3);   // -1 (피연산자 부호 따름)
System.out.println(10 % -3);   // 1
System.out.println(-10 % -3);  // -1
```

## 문자열 연결 (+)

### 문자열 + 문자열

```java
String s1 = "Hello";
String s2 = "World";
String result = s1 + " " + s2;  // "Hello World"
```

### 문자열 + 숫자

```java
String str = "나이: " + 25;      // "나이: 25"
String str2 = "결과: " + 10 + 20;  // "결과: 1020" (왼쪽부터 연결)
String str3 = "결과: " + (10 + 20);  // "결과: 30" (괄호로 연산 먼저)
```

### 연산 순서 주의

```java
System.out.println(1 + 2 + "3");   // "33" (1+2=3, 3+"3"="33")
System.out.println("1" + 2 + 3);   // "123" (문자열 연결)
System.out.println(1 + "2" + 3);   // "123"
```

## 증감 연산자

|연산자|의미|설명|
|---|---|---|
|`++a`|전위 증가|먼저 증가 후 사용|
|`a++`|후위 증가|먼저 사용 후 증가|
|`--a`|전위 감소|먼저 감소 후 사용|
|`a--`|후위 감소|먼저 사용 후 감소|

### 단독 사용 (차이 없음)

```java
int a = 5;
a++;  // a = 6
++a;  // a = 7
a--;  // a = 6
--a;  // a = 5
```

### 수식에서 사용 (차이 있음)

```java
int a = 5;
int b = a++;  // b = 5, a = 6 (후위: 먼저 대입 후 증가)

int c = 5;
int d = ++c;  // d = 6, c = 6 (전위: 먼저 증가 후 대입)
```

### 주의: 복잡한 수식

```java
int a = 5;
int result = a++ + ++a;  // 5 + 7 = 12 (a는 최종 7)
// 가독성이 떨어지므로 권장하지 않음!
```

## Math 클래스 활용

```java
// 절대값
Math.abs(-10);       // 10

// 최대/최소
Math.max(10, 20);    // 20
Math.min(10, 20);    // 10

// 거듭제곱
Math.pow(2, 3);      // 8.0 (2^3)

// 제곱근
Math.sqrt(16);       // 4.0

// 반올림/올림/내림
Math.round(3.6);     // 4
Math.ceil(3.1);      // 4.0
Math.floor(3.9);     // 3.0

// 난수 (0.0 ~ 1.0 미만)
Math.random();       // 0.xxxxx
```

### 난수 활용

```java
// 1 ~ 10 사이 정수
int rand = (int)(Math.random() * 10) + 1;

// 0 ~ 99 사이 정수
int rand2 = (int)(Math.random() * 100);

// min ~ max 사이 정수
int min = 5, max = 15;
int rand3 = (int)(Math.random() * (max - min + 1)) + min;
```

---

> [!tip] 핵심 정리
> 
> - 정수/정수 = 정수 (소수점 버림)
> - `%`로 짝수/홀수, 배수 판별
> - 문자열 `+` 는 연결 연산
> - `++a`(전위)와 `a++`(후위) 차이 주의
> - 복잡한 증감 연산은 피하기
