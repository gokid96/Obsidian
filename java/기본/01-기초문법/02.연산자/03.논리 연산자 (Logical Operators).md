## 개요

- boolean 값을 연산하여 **boolean 결과** 반환
- 조건문에서 여러 조건을 결합할 때 사용

---

## 논리 연산자 종류

|연산자|의미|설명|
|---|---|---|
|`&&`|AND (그리고)|둘 다 true면 true|
|`||`|
|`!`|NOT (부정)|true↔false 반전|

---

## AND 연산자 (&&)

### 진리표

|A|B|A && B|
|---|---|---|
|true|true|**true**|
|true|false|false|
|false|true|false|
|false|false|false|

### 예시

```java
boolean a = true;
boolean b = false;

System.out.println(a && a);  // true
System.out.println(a && b);  // false
System.out.println(b && b);  // false

// 조건문에서 활용
int age = 25;
boolean hasLicense = true;

if (age >= 18 && hasLicense) {
    System.out.println("운전 가능");
}
```

---

## OR 연산자 (||)

### 진리표

|A|B|A \| B|
|---|---|---|
|true|true|true|
|true|false|**true**|
|false|true|**true**|
|false|false|false|

### 예시

```java
boolean a = true;
boolean b = false;

System.out.println(a || a);  // true
System.out.println(a || b);  // true
System.out.println(b || b);  // false

// 조건문에서 활용
String day = "토요일";

if (day.equals("토요일") || day.equals("일요일")) {
    System.out.println("주말입니다");
}
```

---

## NOT 연산자 (!)

### 진리표

|A|!A|
|---|---|
|true|false|
|false|true|

### 예시

```java
boolean flag = true;

System.out.println(!flag);   // false
System.out.println(!false);  // true
System.out.println(!!flag);  // true (이중 부정)

// 조건문에서 활용
boolean isLoggedIn = false;

if (!isLoggedIn) {
    System.out.println("로그인이 필요합니다");
}
```

## 단락 평가 (Short-circuit Evaluation)

### && 단락 평가

- 첫 번째가 **false**면 두 번째는 **평가 안 함**

```java
int x = 0;

// x가 0이면 두 번째 조건 평가 안 함 (0으로 나누기 방지)
if (x != 0 && 10 / x > 1) {
    System.out.println("조건 충족");
}

// 확인
int a = 5;
boolean result = (a > 10) && (++a > 0);
System.out.println(a);       // 5 (++a 실행 안 됨)
System.out.println(result);  // false
```

### || 단락 평가

- 첫 번째가 **true**면 두 번째는 **평가 안 함**

```java
int a = 5;
boolean result = (a < 10) || (++a > 0);
System.out.println(a);       // 5 (++a 실행 안 됨)
System.out.println(result);  // true
```

### 활용: null 체크

```java
String str = null;

// 단락 평가 덕분에 안전
if (str != null && str.length() > 0) {
    System.out.println(str);
}

// str이 null이면 str.length() 실행 안 됨 → NPE 방지
```

## & 와 | (비트/논리 연산자)

### 차이점

|연산자|단락 평가|용도|
|---|---|---|
|`&&`|O|논리 연산 (권장)|
|`&`|X|논리/비트 연산|
|`||`|
|`|`|X|

```java
int a = 5;

// && : 단락 평가 O
boolean r1 = (a > 10) && (++a > 0);
System.out.println(a);  // 5

// & : 단락 평가 X (항상 둘 다 평가)
a = 5;
boolean r2 = (a > 10) & (++a > 0);
System.out.println(a);  // 6 (++a 실행됨)
```

> [!warning] 주의 일반적으로 `&&`와 `||` 사용을 권장합니다. `&`와 `|`는 비트 연산이나 특수한 경우에만 사용하세요.

## 복합 조건

### 여러 조건 결합

```java
int age = 25;
int score = 80;
boolean isVip = true;

// AND와 OR 혼합
if ((age >= 20 && score >= 70) || isVip) {
    System.out.println("입장 가능");
}
```

### 연산자 우선순위

```
! > && > ||
```

```java
// 예시
boolean a = true, b = false, c = true;

// !a || b && c
// (!a) || (b && c)
// false || false
// false
System.out.println(!a || b && c);  // false

// 명확하게 괄호 사용 권장
System.out.println((!a) || (b && c));  // false
System.out.println(!(a || b) && c);    // false
```

## 드모르간 법칙

### 공식

```
!(A && B) == !A || !B
!(A || B) == !A && !B
```

### 활용

```java
int x = 5;

// 원본
if (!(x > 0 && x < 10)) { }

// 드모르간 적용
if (x <= 0 || x >= 10) { }

// 둘은 동일한 조건
```

### 예시: 유효성 검사

```java
String id = "user123";
String pw = "pass";

// 유효하지 않은 경우
if (id == null || id.isEmpty() || pw == null || pw.isEmpty()) {
    System.out.println("입력값이 유효하지 않습니다");
}

// 드모르간 적용 (유효한 경우)
if (!(id != null && !id.isEmpty() && pw != null && !pw.isEmpty())) {
    System.out.println("입력값이 유효하지 않습니다");
}
```

## XOR 연산자 (^)

### 진리표 (배타적 OR)

|A|B|A ^ B|
|---|---|---|
|true|true|false|
|true|false|**true**|
|false|true|**true**|
|false|false|false|

- 둘이 **다르면** true

```java
boolean a = true;
boolean b = false;

System.out.println(a ^ a);  // false
System.out.println(a ^ b);  // true
System.out.println(b ^ b);  // false

// 활용: 둘 중 하나만 선택
boolean optionA = true;
boolean optionB = false;

if (optionA ^ optionB) {
    System.out.println("하나만 선택됨");
}
```

## 실전 예제

### 로그인 검증

```java
String userId = "admin";
String password = "1234";

if (userId != null && !userId.isEmpty() 
    && password != null && password.length() >= 4) {
    System.out.println("로그인 시도");
}
```

### 범위 체크

```java
int score = 85;

// 0 ~ 100 사이
if (score >= 0 && score <= 100) {
    System.out.println("유효한 점수");
}

// 범위 밖
if (score < 0 || score > 100) {
    System.out.println("유효하지 않은 점수");
}
```

### 조건부 기본값

```java
String name = null;
String displayName = (name != null && !name.isEmpty()) ? name : "Guest";
System.out.println(displayName);  // Guest
```

---

> [!tip] 핵심 정리
> 
> - `&&` : 둘 다 true → true (AND)
> - `||` : 하나라도 true → true (OR)
> - `!` : 반전 (NOT)
> - 단락 평가: `&&`는 false면 중단, `||`는 true면 중단
> - null 체크는 `&&` 단락 평가 활용
