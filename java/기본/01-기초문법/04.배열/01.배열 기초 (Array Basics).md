

## 배열이란?

- **같은 타입**의 데이터를 **연속된 공간**에 저장
- **고정 크기**: 생성 후 크기 변경 불가
- **인덱스**: 0부터 시작하는 번호로 접근

```java
int[] scores = {90, 85, 78, 92, 88};
//              [0] [1] [2] [3] [4]
```

## 배열 선언

### 선언 방식

```java
// 권장 (Java 스타일)
int[] arr;
String[] names;

// 가능 (C 스타일)
int arr[];
String names[];
```

### 선언만 하면 null

```java
int[] arr;  // 선언만 → null
// System.out.println(arr[0]);  // NullPointerException!
```

## 배열 생성

### 1. new 키워드로 생성

```java
int[] arr = new int[5];  // 크기 5인 배열 생성
// 기본값으로 초기화: [0, 0, 0, 0, 0]

String[] names = new String[3];
// 참조 타입은 null로 초기화: [null, null, null]
```

### 2. 리터럴로 생성

```java
int[] arr = {10, 20, 30, 40, 50};

String[] fruits = {"사과", "바나나", "오렌지"};
```

### 3. new + 리터럴

```java
int[] arr = new int[]{10, 20, 30};

// 메서드 인자로 전달할 때 유용
printArray(new int[]{1, 2, 3});
```

## 기본값

|타입|기본값|
|---|---|
|byte, short, int, long|0|
|float, double|0.0|

## 배열 접근

### 인덱스로 접근

```java
int[] arr = {10, 20, 30, 40, 50};

// 읽기
int first = arr[0];   // 10
int third = arr[2];   // 30
int last = arr[4];    // 50

// 쓰기
arr[0] = 100;         // [100, 20, 30, 40, 50]
arr[2] = 300;         // [100, 20, 300, 40, 50]
```

### 배열 길이

```java
int[] arr = {10, 20, 30};
int len = arr.length;  // 3 (메서드 아님, 필드!)

// 마지막 요소 접근
int last = arr[arr.length - 1];  // 30
```

### ArrayIndexOutOfBoundsException

```java
int[] arr = new int[5];  // 인덱스 0~4

arr[5] = 10;  // 예외 발생!
arr[-1] = 10; // 예외 발생!
```

## 배열 순회

### for문

```java
int[] arr = {10, 20, 30, 40, 50};

for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### for-each문

```java
for (int num : arr) {
    System.out.println(num);
}
```

### 역순 순회

```java
for (int i = arr.length - 1; i >= 0; i--) {
    System.out.println(arr[i]);
}
```

## 배열과 메모리

### 참조 타입

```java
int[] arr1 = {1, 2, 3};
int[] arr2 = arr1;  // 같은 배열 참조!

arr2[0] = 100;
System.out.println(arr1[0]);  // 100 (같이 변경됨)
```

### 메모리 구조

```
[Stack]          [Heap]
┌──────┐         ┌─────────────┐
│ arr1 │────┬───→│ [1, 2, 3]   │
├──────┤    │    └─────────────┘
│ arr2 │────┘
└──────┘
```

## 배열 복사

### 반복문으로 복사

```java
int[] original = {1, 2, 3, 4, 5};
int[] copy = new int[original.length];

for (int i = 0; i < original.length; i++) {
    copy[i] = original[i];
}
```

### System.arraycopy()

```java
int[] src = {1, 2, 3, 4, 5};
int[] dest = new int[5];

System.arraycopy(src, 0, dest, 0, src.length);
// (원본, 시작위치, 대상, 시작위치, 복사길이)
```

### Arrays.copyOf()

```java
int[] original = {1, 2, 3, 4, 5};
int[] copy = Arrays.copyOf(original, original.length);

// 크기 다르게 복사
int[] bigger = Arrays.copyOf(original, 10);  // [1,2,3,4,5,0,0,0,0,0]
int[] smaller = Arrays.copyOf(original, 3);  // [1,2,3]
```

### clone()

```java
int[] original = {1, 2, 3};
int[] copy = original.clone();
```

## 배열 비교

### == 는 참조 비교

```java
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};

System.out.println(arr1 == arr2);  // false (다른 객체)
```

### Arrays.equals()로 내용 비교

```java
System.out.println(Arrays.equals(arr1, arr2));  // true
```

## 실전 예제

### 합계와 평균

```java
int[] scores = {90, 85, 78, 92, 88};
int sum = 0;

for (int score : scores) {
    sum += score;
}

double avg = (double) sum / scores.length;
System.out.println("합계: " + sum);      // 433
System.out.println("평균: " + avg);      // 86.6
```

### 최대값/최소값

```java
int[] arr = {34, 78, 12, 98, 45};
int max = arr[0];
int min = arr[0];

for (int i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
    if (arr[i] < min) min = arr[i];
}

System.out.println("최대: " + max);  // 98
System.out.println("최소: " + min);  // 12
```

### 특정 값 검색

```java
int[] arr = {10, 20, 30, 40, 50};
int target = 30;
int index = -1;

for (int i = 0; i < arr.length; i++) {
    if (arr[i] == target) {
        index = i;
        break;
    }
}

if (index != -1) {
    System.out.println(target + " 위치: " + index);
} else {
    System.out.println("없음");
}
```

---

> [!tip] 핵심 정리
> 
> - 배열: 같은 타입, 고정 크기, 연속 저장
> - 인덱스: 0부터 시작
> - `length`: 배열 길이 (필드)
> - 배열은 참조 타입 → 대입 시 같은 배열 공유
> - 복사: `Arrays.copyOf()`, 비교: `Arrays.equals()`
