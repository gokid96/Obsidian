### 1. 결합도를 낮추기 위해 (가장 중요)
```java
// ❌ new로 직접 생성 (강한 결합)
@RestController
public class UserController {
    private UserService userService = new UserService();
    // UserController가 UserService 구체 클래스에 강하게 의존
    // UserService 바꾸려면 여기 코드도 수정해야 함
}

// ✅ DI 사용 (느슨한 결합)
@RestController
public class UserController {
    private final UserService userService;
    
    public UserController(UserService userService) {
        this.userService = userService;  // 외부에서 주입받음
    }
}
```
