## 📊 실제 포트폴리오에 많이 등장하는 트러블슈팅

### **1. 성능 개선 (⭐⭐⭐ 가장 인기)**

이미지 리사이징 성능 문제 발생부터 여러 라이브러리 비교 검토 및 AWS CloudFront 선택까지의 과정과, 8000건 대용량 데이터 조회 성능 개선을 위한 캐싱 전략 수립 과정 [개발자 포트폴리오 예시 5개 모음 공유 | 백엔드, 프론트엔드 개발자 포트폴리오 - 내일배움캠프 블로그](https://nbcamp.spartacodingclub.kr/blog/개발자-포트폴리오-예시-5개-모음-공유-백엔드-프론트엔드-개발자-포트폴리오-63589)

**예시:**

```
❌ 문제: API 응답 속도 3초 → 사용자 이탈
🔍 분석: N+1 쿼리 문제 발견
✅ 해결: Fetch Join 적용
📊 결과: 응답시간 3초 → 0.3초 (10배 개선)
```

### **2. 동시성 문제**

InMemory 방식의 사용자 대기 이슈부터 WebSocket과 Redis를 활용한 실시간 세션 관리 및 데이터 무결성 보장 [개발자 포트폴리오 예시 5개 모음 공유 | 백엔드, 프론트엔드 개발자 포트폴리오 - 내일배움캠프 블로그](https://nbcamp.spartacodingclub.kr/blog/개발자-포트폴리오-예시-5개-모음-공유-백엔드-프론트엔드-개발자-포트폴리오-63589)

**예시:**

```
❌ 문제: 동시에 같은 상품 구매 시 재고 오류
🔍 분석: Race Condition 발생
✅ 해결: 비관적 락(Pessimistic Lock) 적용
📊 결과: 재고 오차 0건
```

### **3. 데이터베이스 설계/최적화**

데이터베이스 선택 과정에서 MySQL과 다른 DB의 장단점 비교 분석부터 프로젝트 규모에 맞는 최적 DB 선택 [개발자 포트폴리오 예시 5개 모음 공유 | 백엔드, 프론트엔드 개발자 포트폴리오 - 내일배움캠프 블로그](https://nbcamp.spartacodingclub.kr/blog/개발자-포트폴리오-예시-5개-모음-공유-백엔드-프론트엔드-개발자-포트폴리오-63589)

**예시:**

```
❌ 문제: 댓글 조회 시 게시글 정보 중복 조회
🔍 분석: 연관관계 매핑 및 쿼리 비효율
✅ 해결: DTO 프로젝션 + 인덱스 추가
📊 결과: 쿼리 수 50% 감소
```

## 💎 특히 좋은 트러블슈팅 주제들

### **백엔드 관점:**

**🔥 1. 성능/최적화 (가장 인기)**

- N+1 문제 해결
- 쿼리 최적화 (인덱스 추가)
- 캐싱 전략 (Redis)
- 페이징 성능 개선
- 이미지/파일 업로드 최적화

**🔐 2. 동시성/트랜잭션**

- 재고 관리 동시성 제어
- 좋아요 중복 방지
- 분산 락 구현
- 트랜잭션 격리 수준 조정

**🗄️ 3. 데이터베이스**

- 대용량 데이터 처리
- 복잡한 연관관계 설계
- 인덱스 전략
- 쿼리 튜닝

**🌐 4. API/서버**

- 대용량 트래픽 처리
- Rate Limiting 구현
- API 응답 시간 개선
- 예외 처리 체계화

**☁️ 5. 배포/인프라**

- CI/CD 파이프라인 구축
- 무중단 배포
- 서버 비용 최적화
- 모니터링 시스템 구축

## ✅ 좋은 트러블슈팅 작성법

### **구조:**

```
1. 문제 정의 (왜 문제인가?)
   - 수치로 표현 (응답시간 3초, 에러율 5%)
   
2. 원인 분석 (어떻게 찾았나?)
   - 디버깅 과정
   - 모니터링 도구 활용
   - 가설 수립
   
3. 해결 방안 탐색 (여러 선택지 고려)
   - A안: 장점 / 단점
   - B안: 장점 / 단점
   - 최종 선택한 이유
   
4. 구현 (어떻게 적용했나?)
   - 핵심 코드 일부
   - 적용 과정의 어려움
   
5. 결과 측정 (개선 효과)
   - Before/After 수치 비교
   - 부가적 효과
```

## 🎯 당신의 게시판 프로젝트에 추천하는 트러블슈팅

**1순위:**

- **N+1 문제 해결** (댓글 조회 시 발생)
- **페이징 성능 개선** (게시글 목록)

**2순위:**

- **동시성 제어** (좋아요/조회수)
- **쿼리 최적화** (검색 기능)

**3순위:**

- **캐싱 적용** (인기 게시글)
- **파일 업로드 최적화** (이미지)

**가장 현실적이고 설득력 있는 트러블슈팅을 1-2개만 깊게 파는 게 좋아**
