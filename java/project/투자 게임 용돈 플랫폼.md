### **🎯 서비스 정의**

**"실제 주식 데이터로 투자 게임을 하고, 성과에 따라 용돈을 받는 게임화된 금융교육 플랫폼"**

---

### **💡 핵심 메커니즘**

**본질**: 게임 성과 기반 용돈 지급 시스템

```
1. 게임머니 지급: 매월 10만원 가상머니
2. 실제 주가로 투자 게임: 진짜 주식 데이터 활용
3. 성과 기반 용돈: 수익 달성 시에만 실제 용돈 지급
4. 리스크 교육: 손실 시 용돈 없음
```


**환산 공식**:

```
게임머니 5만원 수익 = 실제 5,000원 용돈 (1/10 환산)
```

### **👥 타겟 사용자**

**주 사용자**: 초등 고학년 ~ 고등학생 (10~18세) **의사결정자**: 자녀 금융교육에 관심 있는 부모 **페인 포인트**: 기존 금융교육은 지루하고 비현실적

### **📱 핵심 기능**

**자녀용 앱**

```

🎯 용돈 요청: 수익 달성 시 실제 용돈 요청

💸 용돈 승인: 자녀 성과 기반 요청을 받아 용돈 지급 결정

📈 교육 효과: 자녀 금융 학습 진행도 확인
```

### **🎯 핵심 가치 제안**

**솔직한 포지셔닝**:

> "결국 부모가 주는 용돈이지만,  
> 가장 재미있고 교육적인 방식으로 받는 앱"

**아이에게**:

- 지루한 공부 대신 재미있는 투자 게임
- 성과에 따른 실질적 보상
- 자연스러운 경제 개념 학습

**부모에게**:

- 자동화된 자녀 금융 교육
- 단순 용돈보다 교육적 효과
- 자녀와 경제 대화의 새로운 소재

---

### **📊 예상 사용 시나리오**

**성공 케이스**:

```
1월: 삼성전자 매수 → 15% 수익 → 1,500원 획득
2월: 애플 분산 투자 → 8% 수익 → 800원 획득  
3월: 급락장 → 20% 손실 → 용돈 없음 (리스크 학습)
4월: 신중한 투자 → 12% 수익 → 1,200원 획득
```

**교육 효과**: 단순 암기가 아닌 체험으로 경제 원리 학습

---

### **🚀 차별화 포인트**

1. **실제 시장 데이터**: 진짜 주식 시장과 연동
2. **게임화**: 재미있는 UX/UI로 높은 참여율
3. **실질적 보상**: 가상이 아닌 실제 용돈 지급
4. **교육적 효과**: 체험 기반 금융 학습
