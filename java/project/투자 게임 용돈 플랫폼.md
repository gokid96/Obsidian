📱 수익 연동 용돈 시스템

1️⃣ 기본 메커니즘
- 아이가 모의투자에서 목표 달성 시
- 부모에게 "용돈 요청" 알림 발송
- 부모가 승인하면 실제 용돈 지급

2️⃣ 용돈 요청 트리거 (부모가 설정)
✅ 수익률 목표 달성 (예: +10% 달성)
✅ 연속 수익일 달성 (예: 5일 연속 플러스)
✅ 투자 미션 완료 (예: "배당주 5개 찾기")
✅ 금융 퀴즈 통과 (예: "10문제 연속 정답")
✅ 일정 기간 포트폴리오 유지 (장기투자 교육)

3️⃣ 용돈 지급 방식
- 앱 내 간편송금 (퍼핀/아이부자처럼)
- 연동된 자녀 계좌로 송금
- 가상 "용돈 포인트" → 나중에 현금화

4️⃣ 부모 설정 기능
- 월 최대 용돈 한도 설정
- 요청 가능 횟수 제한
- 난이도 조절 (목표 수익률 설정)
- 보상 금액 자동/수동 설정

```
## 🎮 게이미피케이션 + 용돈 시스템 결합 예시

### "투자 레벨업 → 용돈 업그레이드"
```
📊 레벨 시스템
- Lv.1 주식 새싹: 기본 용돈 1,000원
- Lv.5 주식 모종: 보너스 용돈 +500원
- Lv.10 투자 나무: 보너스 용돈 +1,000원
- Lv.20 투자 전문가: 특별 보너스 5,000원
```
### "미션 완수 → 즉시 용돈"
```
🎯 일일 미션
✅ 오늘의 뉴스 읽기 → 100원
✅ 3개 기업 분석하기 → 200원
✅ 포트폴리오 점검하기 → 100원

🏆 주간 미션
✅ 5일 연속 접속 → 500원
✅ 수익률 +5% 달성 → 1,000원
✅ 금융 퀴즈 10개 맞히기 → 700원
```

## ⚠️ 주의할 점

### 1. **과도한 보상 방지**
- 부모가 한도 설정 가능하게
- "투자 = 돈 버는 것"이 아닌 "공부하면 보상"으로 프레이밍
- 손실 시에도 학습 보상 (과정 중시)

### 2. **사행성 우려 회피**
```
❌ 피해야 할 것:
- "수익률 높을수록 용돈 많이" (도박처럼 느껴짐)
- 실시간 용돈 요청 알림 남발

✅ 권장하는 방식:
- "목표 달성 + 학습 완료" 모두 충족
- 주 1회 정기 정산일 설정
- 과정과 노력에 대한 보상 강조
```

### 3. **교육적 가치 유지**
```
수익 달성 시:
"축하해요! 🎉 
삼성전자가 10% 올랐네요.
왜 올랐는지 알아볼까요?"

→ 단순 수익 → 용돈이 아닌
→ 학습 → 이해 → 보상 구조
