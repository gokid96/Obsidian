1. **컨트롤 노드에서 타겟 노드로 SSH접속을 위한 SSH키 생성**
	- `ssh-keygen -t rsa -b 2048` 명령어로 공개키/개인키 생성 
	- 개인키는 컨트롤 노드에 보관 (~/.ssh/id_rsa)
	- 공개키는 타겟 노드에 배포 (~/.ssh/id_rsa.pub)

2. **타겟노드에 SSH키 설정**
	- `ssh-copy-id` 명령어로 공개키를 타겟 서버에 복사
	- 타겟 노드의 `~/.ssh/authorized_keys` 파일에 공개키 저장

3. **앤서블 설치**
	- 컨트롤 노드에만 설치하면 됨 (타겟 노드에는 설치 불필요)
	- SSH를 통해 원격 서버를 관리하는 도구

4. **작업 디렉토리 생성** 
	-  (/home/sungmin/ansible-test)
	- 인벤토리 파일, 플레이북, 설정 파일 등을 저장
	- 프로젝트별로 Ansible 작업을 구조화하여 관리

5. **인벤토리 파일 생성**
	  (/home/sungmin/ansible-test/hosts)
	   [webservers]
	  192.168.14.32
	  192.168.14.33
	  192.168.14.34

	- 관리할 서버 목록과 그룹 정의
	- 어떤 서버들을 관리할지 명시

	- 접속 정보 설정 (사용자명, 포트 등)
